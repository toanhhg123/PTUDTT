## Cài đặt và chạy thử nghiệm
1. **Yêu cầu hệ thống:**
   - Node.js và npm
   - .Net 8.0 SDK
   - SQL Server
   - Expo server
   - React, React Native
   - Docker

2. **Cài đặt:**
- Bắt buộc phải cài đặt .NET 8.0 SDK
- Clone repository
- Vào đường dẫn chứa file program.cs
- Chạy theo từng bước:

 ```bash
dotnet restore
dotnet build
dotnet run
```
